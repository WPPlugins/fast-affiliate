== Description ==

Fast affiliate is a wordpress plugin to make easier affiliate links addition in your posts. It's works with affiliation platforms API's to search products and generate affiliate links. This plugin manages products availability too, and you can add your affiliate products in bottom of post and in a widget.
- search in your products from your affiliates programs
- place in one click the affiliate product in your post
- control on one page which link doesn\'t exist anymore
- control on one page which link is out of stock

Fast Affiliate is available in English, Spanish and French

== Installation ==

This section describes how to install the plugin and get it working.
1. Download and install the plugin from WordPress dashboard. You can also upload the entire “Fast Affiliate” folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the ‘Plugins’ menu in WordPress
3. Go to post config section, and add your affiliate api keys.
